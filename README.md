# t2s

Toporobot to Survex

## perl script: t2s

Convert Toporobot exported TEXT file into Survex file format (.svx)

Requires Perl 5

```bash session
$ perldoc t2s

$ t2s toporobot_TEXT_file
```
## Text Editor vim 

Edit file again, using mac file format
```vim
:e ++ff=mac
```

This buffer will use CR-only line endings when written
```vim
:setlocal ff=mac
```
